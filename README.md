# AI-Powered Property Sales Assistant

This capstone project involves the design, development, and deployment of an AI-powered assistant for real estate sales professionals on the Salesforce Platform. The solution will leverage Agentforce, Flow Builder, and other Salesforce AI capabilities to streamline property inquiry management, provide instant access to property information, and automate routine sales tasks. The project aims to enhance agent efficiency and improve client engagement through intelligent automation.
